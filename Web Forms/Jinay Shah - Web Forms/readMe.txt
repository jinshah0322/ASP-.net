Steps of implementation

1) Run the EmployeeDBScript file to generate the Database which contains all the tables.
2) Open the EmployeeManagement folder which will contain EmployeeManagement.sln file in visual studio.
3) Start the project by clicking start button or  with "ctrl+F5" command in visual studio.
4) On running the project you will be redirected to https://localhost:44351/Default page where there will be button to navigate to other pages and access different webforms.
